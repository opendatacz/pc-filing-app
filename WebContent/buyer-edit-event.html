On this screen, you can edit an event.
Enter a title for the event and at least one CPV code saying what will be the object of the event.
You don’t need to know the exact code, just start typing either the CPV code or its label and let autocomplete help you.
When you select the event type, you will be asked to enter more information according to the type.
Mandatory fields have a red asterisk next to them.
